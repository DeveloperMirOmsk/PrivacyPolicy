<h2 align="center">3.	ПЕРЕЧЕНЬ СОБИРАЕМЫХ ДАННЫХ И ЦЕЛИ ОБРАБОТКИ</h2>

###

<p align="left"> 
<strong> 3.1.</strong> 	Правообладатель не собирает и не обрабатывает данные, позволяющие идентифицировать личность Пользователя.<br>
<strong>3.2.</strong>	В связи с использованием Мобильного приложения Правообладатель может автоматически собирать и обрабатывать следующую информацию, которая не может быть использована для прямой идентификации Пользователя:<br>
<strong>3.2.1.</strong> Данные от приборов учета (показания, серийный номер и т.п.) для сохранения и отображения показаний, ведения истории потребления ресурсов.<br>
<strong>3.2.2.</strong>	Информация о типе мобильного устройства, версии операционной системы, версии приложения.<br>
<strong>3.3.</strong>	Цели обработки данных:<br>
<strong>3.3.1.</strong>	Отображение текущих показаний и истории потребления ресурсов Пользователю.<br>
<strong>3.3.2.</strong>	Идентификация приборов учета в системе Правообладателя.<br>
<strong>3.3.3.</strong>	Проведение аналитических исследований на основе обезличенных данных с целью совершенствования работы Приложения, а также повышения качества и стабильности функционирования поддерживаемых приборов учета.<br>
<strong>3.4.</strong>	Для обеспечения функциональности Приложение запрашивает доступ к следующим функциям Устройства:<br>
<strong>3.4.1.</strong>	Доступ к Bluetooth используется исключительно для подключения к приборам учета в рамках протокола Bluetooth Low Energy (BLE). Без данного разрешения использование Приложения по прямому назначению невозможно.<br>
<strong>3.4.2.</strong>	Доступ к геолокации требуется исключительно для поиска приборов учета через Bluetooth на устройствах платформы Android (с версией ниже 10.0). Приложение не собирает и не хранит данные о местоположении пользователя. Эта информация используется только для обнаружения ближайших устройств.<br>
<strong>3.4.3.</strong>	Доступ к сети необходим для передачи показаний и данных приборов учета в систему Правообладателя с целью их обработки и хранения.
